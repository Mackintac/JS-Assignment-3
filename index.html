About this Assignment: REST Countries API

The API that I will be demonstrating for this assignment is the REST Countries API available at: https://restcountries.com/.

Select any of the buttons to update the example queries and explore their respective response objects!

Get All Countries.

This query retrieves information about all countries available in the REST Countries API.

Search for a country by Name.

Search for a country by Full Name.

By providing the exact full name of the country, we can retrieve detailed information about it.

Search for a country by Currency.

By specifying the currency code, we can find countries that use that currency.

Search for a country by Code.

By using the country code, we can retrieve information about a specific country.

Search for a country by Demonym.

By providing the demonym, we can find countries associated with that term.

Search for several countries by Language.

By specifying a language, we can find countries where that language is spoken.

Search for a country by Capital.

By providing the capital city, we can retrieve information about the corresponding country.

Search for several countries by Region.

By specifying a region, we can find all countries located in that region.

Search for several countries by Subregion.

By specifying a subregion, we can find all countries located in that subregion.

Get Borders of a Country.

This query retrieves the list of countries that share a border with the selected country.
